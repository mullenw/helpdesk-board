Helpdesk Ticket Board

This project is a simple Helpdesk Ticket Board built with Next.js, React, and Tailwind CSS.  
Users can view helpdesk tickets, filter by status or priority, search by keyword, and add tickets to a queue.

How to run:

1. Open the project in VS Code.  
2. In the terminal, install dependencies:
   ```bash
   npm install

To start the development server:

npm run dev

Open your browser and go to:
http://localhost:3000
